# Telegram Bot Simulator - Frontend

React-da qurilgan Telegram bot simulator UI.

## O'rnatish

```bash
npm install
```

## Ishga tushirish

```bash
npm start
```

Browser `http://localhost:3000` da ochiladi

## Komponentlar

### ChatUI.jsx
- Foydalanuvchi tanlash
- Xabar yuboring/qabul qiling
- Real-time chat interface

### Styling
- Modern UI design
- Responsive layout
- Mobile-friendly

## Environment Variables

`.env` faylni yarating:

```
REACT_APP_SIMULATOR_URL=http://localhost:8000
```

## Features

✅ Mock users load qilish  
✅ User select qilish  
✅ Message send/receive  
✅ Chat history  
✅ Real-time updates  
✅ Error handling  

## Build

```bash
npm run build
```

Build folder production-ready assets ga javob beradi.
